# Standard Feature Names

| Feature Name | Description |
|---|---|
| `patient_id` | Unique identifier for the patient |
| `age` | Age of the patient in years |
| `sex` | Biological sex of the patient |
| `bmi` | Body Mass Index (kg/m²) |
| `systolic_bp` | Systolic blood pressure (mmHg) |
| `diastolic_bp` | Diastolic blood pressure (mmHg) |
| `glucose` | Blood glucose level (mg/dL) |
| `cholesterol` | Total cholesterol level (mg/dL) |
| `creatinine` | Serum creatinine level (mg/dL) |
| `diabetes` | Diabetes diagnosis indicator (0: No, 1: Yes) |
| `hypertension` | Hypertension diagnosis indicator (0: No, 1: Yes) |
| `diagnosis` | Primary medical diagnosis |
| `readmission_30d` | Readmission within 30 days indicator (0: No, 1: Yes) |
| `mortality` | Patient mortality indicator (0: Alive, 1: Deceased) |
